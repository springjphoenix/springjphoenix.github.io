# 总结

由于东西太多了， 目前并没有整理完， 定期整理资料是一个好习惯， 删除东西真的很爽，因为删的都是自己已经了解不需要了的东西。

不要只做资料收藏着， 要转化为自己的知识。

另外也可以看出， 很多以前花了很多时间研究的东西， 其实早就被淘汰了， 所以很多时候我们不要太纠结技术， 而要关注解决问题本身！

因为， 终究， 所有技术都会死的， 我们也是。。。
